# Method

The HTTP method used to trigger the workflow. This defines the type of request
that will initiate the workflow (e.g., GET, POST).



## Values

| Name     | Value    |
| -------- | -------- |
| `GET`    | GET      |
| `POST`   | POST     |
| `PUT`    | PUT      |
| `PATCH`  | PATCH    |
| `DELETE` | DELETE   |
